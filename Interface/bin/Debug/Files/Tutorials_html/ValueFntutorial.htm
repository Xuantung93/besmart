For example, the criterion "Interaction with user" should be maximized. Assuming we have three software A, B and C, with values of "Interaction with user" equal to [1, 3, 5], the value of the priorities for this criterion would be [0, 0.5, 1].

Another example, the criterion "Cost" should be minimized. For the three software A, B and C, with values of "Cost" equal to [100, 800, 1000], the value of the priorities for this criterion would be [1, 0.222, 0].

 

After, is performed the normalization of the values (so that the sum is equal to 1).
